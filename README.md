# Internal Asset Borrowing Request System

A complete Power Platform / Dynamics 365 solution for managing internal company assets and borrowing requests.

## Technologies

- Microsoft Dataverse
- Model-Driven App
- Business Process Flow
- Business Rules
- JavaScript
- Power Automate
- Classic Workflow
- Security Roles
- Dashboards & Charts

## Features

- Asset Management
- Borrowing Request Lifecycle
- Approval / Rejection Process
- Asset Availability Validation
- Automated Notifications
- Security Role Management
- Dashboard Reporting

## Business Process

Create Asset
→ Create Borrowing Request
→ Submit
→ Approve / Reject
→ Return Asset

## Bonus Features

- Auto Calculate Borrowing Days
- Borrowing Requests by Asset Type Chart
- Asset Borrowing History Notification
- Custom Theme and Branding
